HTML Sectioning

Tag Description
<header> Logo, introductory content, navigational aids, search form, author's name, user login, etc.
<nav> Navigation links, menus, tables of contents, and indexes
<main> Dominate content area of a webpage
<h1> Section headings one to six
<section> Generic standalone section of a document, doesn't have a more specific sectioning element, should include a heading
<article> Forum post, magazine or newspaper article, blog entry, product card, user-submitted comment, interactive widget or gadget, any other independent item of content
<aside> Only indirectly related to main content, sidebars, or call-out boxes
<footer> Footer for the nearest sectioning root element, author information, copyright data, links to related documents
<address> Contact information

CSS Positioning

Property & Value Description
position: static; Default positioning. Not affected by top, bottom, left, and right properties.
position: absolute; Positioned relative to nearest positioned ancestor. Without positioned ancestor, uses document body and scrolls with page. Can overlap other content. Uses top, bottom, left, and right properties.
position: relative; Positioned relative to default. Other content will still respect the default positioning. Can be positioned with top, bottom, left, and right properties.
position: fixed; Fixed in place relative to user viewpoint. Does not leave a default positioning gap. Positioned with top, bottom, left, and right properties.
position: sticky; Static position until user scrolls past predetermined offset and which point it "sticks" in a fixed position. Requires at least one top, bottom, left, or right property.

CSS Z-Index

Property & Value Description
z-index: 100; Specifies which elements are in the front or back. Uses positive or negative numbers with the largest number in the front. Does not work with static positioning.

CSS Display

Property & Value Description
display: inline; Will not start a new line and only takes up the minimum necessary space. Example default inline elements: <span> <a> <img>
display: block; Will start a new line and takes up the full width available. Example default block elements: <div> <h1> <p> <form> <header> <footer> <section>
display: inline-block; Will not start a new line and takes up a determined width and height. Also respects top and bottom margins and padding.
display: none; Used with JS to hide and show elements without deleting them. Example default display none: <script>
visibility: hidden; Will also hide an element without affecting page layout.

CSS Float & Clear

Property & Value Description
float: left; Positions content within a container. Uses the values left, right, inherit, and none (default). Can be used to position multiple <div> type elements next to each other.
clear: left; Determines what should happen with elements next to floating elements.
  • none; - Does not push element below left or right floating elements (default)
  • left; - Pushes element below left floating element
  • right; - Pushes element below right floating element
  • both; - Pushes element below left and right floating elements
  • inherit; - Element inherits clear value from its parent
